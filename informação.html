Sobre o Sistema de Saúde

Este sistema foi criado para facilitar a ligação entre pacientes e médicos, permitindo gestão de consultas, organização de dados e melhor atendimento.

👨‍⚕️ Para Médicos

Os médicos podem visualizar pacientes, acompanhar consultas e gerir o atendimento.

🧑‍🦱 Para Pacientes

Os pacientes podem marcar consultas, ver histórico e acompanhar os seus atendimentos.

🔐 Segurança

O sistema separa médicos e pacientes para garantir organização e privacidade.
